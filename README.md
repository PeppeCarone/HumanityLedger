# HumanityLedger

Gioco di simulazione politica narrativa in Godot 4.6. Il giocatore guida un popolo attraverso le ere della storia, prendendo decisioni con interazioni gestuali (drag-and-drop) che modificano in tempo reale le sorti della civiltà.

Progetto universitario sviluppato da 2 persone.

---

## Stato

Pre-alpha. Documentazione di design completata, prototipo in costruzione. Vedi `Docs/06-roadmap.md` per la timeline.

## Come avviare

### Requisiti

- [Godot Engine 4.6](https://godotengine.org/download) stable
- Windows / macOS / Linux (build ufficiale solo Windows)

### Setup

```bash
git clone https://github.com/PeppeCarone/HumanityLedger.git
cd HumanityLedger
```

Poi apri `project.godot` con Godot 4.6 e premi F5.

> Il file `project.godot` sarà presente dalla settimana 1 di sviluppo. Al momento il repo contiene solo documentazione.
